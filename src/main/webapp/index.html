Ha ocurrido un error 😒

posibles causas

  1. Iniciaste la aplicación desde un IDE y no la ejecutaste npm start or npm run webapp:build.
  2. Se produjo un error de red al ejecutar npm install. Si estás conectado a través de un proxy corporativo, es probable que este error haya sido causado por tu proxy.
  3. Has instalado una versión de Node.js que no es compatible con Nuestra App: por favor, utiliza una versión LTS (soporte a largo plazo), ya que es la única versión que admitimos.

Recompilar el código del lado del cliente

Si quieres ir rápido, ejecuta ./npmw run app:start para crear y ejecutar todo.

Si quieres tener más control para poder depurar tu problema más fácilmente, debes seguir estos pasos:

  1. Instala las dependencias de npm con el comando npm install
  2. Compila el cliente con el comando npm run webapp:build o npm start
  3. Inicia el servidor con ./npmw run app:start o utilizando tu IDE
